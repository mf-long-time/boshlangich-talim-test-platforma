# Online test generator

Bu papkada `index.html` fayli mavjud. Uni GitHub Pages orqali online ishga tushirish mumkin.

## GitHub Pages'ga joylash

1. GitHub profilingizga kiring.
2. New repository bosing.
3. Repository nomini kiriting, masalan: `talim-test-platforma`.
4. Public tanlang.
5. `index.html` faylini yuklang.
6. Repository ichida Settings → Pages bo‘limiga kiring.
7. Branch: `main`, folder: `/root` tanlang va Save bosing.
8. Bir necha daqiqadan keyin link ochiladi:
   `https://username.github.io/talim-test-platforma/`

## Platforma imkoniyatlari

- mavzu tanlash;
- hammasidan aralash test;
- savollar sonini tanlash;
- savollarni tasodifiy aralashtirish;
- javob variantlarini tasodifiy aralashtirish;
- oxirida faqat natija va foizni ko‘rsatish;
- mobil qurilmada ishlash.
